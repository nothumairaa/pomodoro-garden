# pomodoro-garden
A focus timer that grows a living garden — one session at a time.

Every time you complete a focus session, a new plant blooms in your garden. The longer you focus, the rarer and more spectacular the flower.

| Session length | Plant reward | Rarity |
|---|---|---|
| 15 min | Seedling / sprout | Common |
| 25 min | Wildflower (poppy, daisy…) | Uncommon |
| 45 min | Sunflower | Rare |
| 60 min | Exotic orchid | Exotic |
 
Custom durations are supported and scale proportionally.
 
---

## ✨ Features
 
- 🕐 **Working Pomodoro timer** with progress ring, pause/resume, and reset
- 🌻 **4 plant tiers** with unique SVG illustrations and colour variants
- 🪴 **Persistent garden** — your plants survive page reloads (localStorage)
- 🦋 **Butterfly rewards** for 7+ day streaks
- ☁️ **Ambient animations** — clouds, birds, swaying plants
- 📊 **Stats bar** — total plants, focus hours, streak, rare count
- 🖱 **Hover tooltips** on each plant (name, rarity, date, duration)
- ⌨️ **Keyboard shortcuts** — `Space` to start/pause, `R` to reset
- 📱 **Responsive** — works on mobile and desktop
- 🚫 **No dependencies** — single HTML file, no build step, no accounts
---

## 🎨 Tech
 
- Vanilla HTML, CSS, and JavaScript — zero dependencies
- All plant illustrations are procedurally generated SVGs
- Data persists via `localStorage` — no server required
- Fonts: [Lora](https://fonts.google.com/specimen/Lora) + [DM Mono](https://fonts.google.com/specimen/DM+Mono) (Google Fonts)
